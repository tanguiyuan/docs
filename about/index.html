# 博客搭建

# 前言

本文将教导你如何安装vuepress搭建一个私人博客，以及如何部署到github上

# 准备工作

下载nodejs和npm,nodejs自带npm nodejs:nodejs

# vuepress概述

  • Vue 驱动的静态网站生成器

  • 基于markdown语法生成网页

  • 可自定义和扩展样式

  • 可以发布至github

  • vuepress中文文档：vuepress

# github仓库

# 初始化仓库

复制ssh链接，第一次使用会提示添加ssh key,百度github ssh key

在D盘新建一个目录work,地址D:/work

进入work目录，右键点击git bash Here，如果没有那么你没有下载git，是无法使用的，百度下载git

git clone git@github.com:tanguiyuan/docs.git
1

把git clone后面的改为自己的

如果第一次使用git需要添加git config user.name和user.email

git config user.name "name"
git config user.email "xx@qq.com"
1
2

然后会出现docs目录、.gitignore文件和README.md文件，到了这一步先不管。

# github部署

  • 在package中添加scripts脚本

    {
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs",
    "deploy": "bash deploy.sh"
  }
}
    1
    2
    3
    4
    5

  • 在docs通目录下添加deploy.sh脚本

    # 构建
vuepress build docs

# 导航到构建输出目录
cd docs/.vuepress/dist

git init
git add -A
git commit -m 'deploy'

# 推到你仓库的的 gh-page 分支
# 将 <USERNAME>/<REPO> 替换为你的信息  
git push -f git@github.com:<USERNAME>/<REPO>.git master:gh-pages
